# MEDI ALPHA PLUS 採用LP

医療専門輸送パートナー「MEDI ALPHA PLUS」の採用ランディングページ（LP）です。  
軽貨物ドライバー（医療検体輸送）の採用応募最大化を目的とした、GitHub Pages 公開対応の静的サイトです。

## デザイン方針

- **カラー**: 白ベース × ネイビー × 青アクセント × ゴールドポイント
- **フォント**: Noto Serif JP（見出し）、Noto Sans JP（本文）、Inter（英数字）
- **トーン**: 清潔感・信頼感・高級感・医療専門性
- **レスポンシブ**: PC / タブレット / スマートフォン 完全対応

---

## JavaScript 機能

- ヒーロー背景：ユーザー提供の集合写真を優先表示（フォールバックあり）
- ヘッダー：スクロールで白背景に切り替わるスティッキーヘッダー
- ハンバーガーメニュー：モバイル対応
- フェードインアニメーション：Intersection Observer によるスクロール連動
- 数値カウントアップ：ヒーロー統計エリア
- 共感カードの選択トグル
- 固定CTAバー：ヒーロー通過後に下部に表示

---

## GitHub Pages 公開手順

1. GitHubに新規リポジトリを作成（例: `medialphaplus-recruit`）
2. 全ファイルをプッシュ
3. リポジトリの **Settings → Pages**
4. **Branch: main** を選択して Save
5. 数分後に公開URL発行
